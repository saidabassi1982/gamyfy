# The Betting Grid Phone Companion

Public PeerJS companion page for The Betting Grid.

Main game QR links should point here:

```text
https://saidabassi1982.github.io/gamyfy/betting-grid/?code=ROOM
```

The host game creates a PeerJS room with the `btgd-` prefix. This page joins that room, submits private numeric guesses, and sends up to two wager chips per betting phase.
